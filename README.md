# pascals_triangle_ruby

Note that the row index starts from 0 (5 row => row[5] ; 5 rows => row.first(5) => rows[0..4]).

## Usage examples

Input:

`ruby pascals_triangle_app.rb 5 row`

Output:
```
[1, 5, 10, 10, 5, 1]
```

Input:

`ruby pascals_triangle_app.rb 5 rows`

Output:
```
[1]
[1, 1]
[1, 2, 1]
[1, 3, 3, 1]
[1, 4, 6, 4, 1]
```
